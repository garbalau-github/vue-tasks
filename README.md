# vue-tasks

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Get access to json data

```
yarn backend 
```

Now, under localhost:8080 you have Vue frontend, and on localhost:5000/tasks you have backend json
